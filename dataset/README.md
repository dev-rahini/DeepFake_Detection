# Dataset

This project uses the ""FaceForensics++ (C23)"" dataset for deepfake detection.

## Dataset Source

Download the dataset from Kaggle:

https://www.kaggle.com/datasets/xdxd003/ff-c23

## Folder Structure

After downloading and extracting, the dataset should have the following structure:

```
FaceForensics++_C23/
├── original/
├── DeepFakeDetection/
├── Deepfakes/
├── Face2Face/
├── FaceShifter/
├── FaceSwap/
├── NeuralTextures/
└── csv/
```

## Dataset Information

* Total Videos: 7,000
* Real Videos: 1,000
* Deepfake Videos: 6,000
* Compression: C23 (H.264, CRF 23)

## Note

The dataset is **not included** in this repository because of its large size and Kaggle's distribution policy. Please download it separately using the link above.
